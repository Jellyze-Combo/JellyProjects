# JellyProjects
A repository of ideas

# 果冻计划
这是一个存放想法的仓库，包含未实现的想法，已完成的想法和在进行中的想法。

## 已完成想法
已完成的想法会以独立仓库的形式作为单独项目存储，本仓库中的已完成想法文件夹存放指向仓库的文本文件。

## 进行中想法
进行中的想法会单独以文本文件的形式存放，其中每个想法会记录开始的想法，实现的进度等。

## 未完成想法
未完成的想法是因能力有限或时间有限暂时未能开始的想法，每个文件记录一个未来可能实现的想法或思路，供自己记录和启发他人。


**无论什么时候，都不要放弃你的任何想法，说不定以后就实现了呢！？**
